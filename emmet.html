HTML

Ajouter un enfant : ">"

li>p
ul>li>p

Ajouter un frère : "+"

ul>li>p+a

Maltiplier : "*"

ul>li*3>p+img

Ajouter une class

ul.wrapper .container

Grouper des éléments Maltipliés : "()"

ul>(header>ul>li*2)+footer>p ul>(li>p+img+a)+li*2>p+img

Insérer du texte : "{}"

a{cliquer} p>{cliquer}+a{ici}+{pour continuer}

Ajouter une numérotation : "$"

ul>li.item$*5

Numéroter à partir d'un chiffer au choix : "$@x"

ul>li.item$@3*5

Numéroter dans l'ordre décroissant : "$@-"

ul>li.item$@-{pizza $@-}*5

Remonter au parent : "^"

div>p>em{hello}^a

Ajouter un attribut à une balise : "[]"

img[src="http://20minutes.fr" alt="site infos"]

Ajouter un id

nav#id1.coco.azerty
